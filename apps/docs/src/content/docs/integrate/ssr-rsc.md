Snapshot has dedicated SSR and Vite surfaces for server rendering and app integration.

Use this path when you need:

- file-based React SSR
- manifest-driven SSR
- RSC-aware rendering
- prefetch manifest generation
- static params / prerender support

Start with these references:

- [SSR Reference](/reference/ssr/)
- [Vite Reference](/reference/vite/)

Canonical source for this surface lives in:

- `src/ssr/index.ts`
- `src/ssr/render.ts`
- `src/ssr/renderer.ts`
- `src/ssr/manifest-renderer.ts`
- `src/ssr/rsc.ts`
- `src/vite/index.ts`
