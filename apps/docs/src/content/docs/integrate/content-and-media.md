Use this path when your app needs rich content creation or media presentation.

Snapshot currently includes:

- markdown and code rendering
- rich input and rich text editing
- file upload flows
- link embeds and compare views
- image, video, carousel, and embed components
- emoji and GIF selection that can be composed into communication experiences

Start with these source-backed references:

- [UI Reference](/reference/ui/)
- [Component Catalog](/reference/components/)
- [Manifest Reference](/reference/manifest/)
- [Capabilities](/start-here/capabilities/)

Canonical source for this surface lives in:

- `src/ui/components/content/**`
- `src/ui/components/media/**`
- `src/ui/components/communication/emoji-picker/**`
- `src/ui/components/communication/gif-picker/**`
- `src/ui/manifest/schema.ts`
- `playground/src/showcase.tsx`

Use the [Examples and Showcase](/examples/) page when you want to see the current component coverage that Snapshot ships on `main`.
