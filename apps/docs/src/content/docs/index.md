Snapshot is the frontend counterpart to Bunshot. It supports two primary ways of building:

- **Manifest-first apps** for teams that want app assembly driven from `snapshot.manifest.json`
- **SDK-driven apps** for teams that want `createSnapshot`, hooks, generated API clients, and custom React

Snapshot also includes:

- SSR and manifest rendering for Bunshot-backed apps
- RSC-aware SSR infrastructure
- Vite plugins for app bootstrapping, sync, and SSR
- a large config-driven UI catalog
- CLI scaffolding and sync flows
- community, realtime, push, and content components

Use the pages under **Start Here** to choose the right path quickly.
